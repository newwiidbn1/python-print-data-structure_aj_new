# Python Grader — Print Data Structure

เว็บแบบฝึกหัด Python สำหรับนักศึกษา ประกอบด้วย 21 ข้อย่อย รวม 20 คะแนน

## โครงสร้างคะแนน

- Numeric 6 ข้อ ข้อละ 1 คะแนน = 6 คะแนน
- String 4 ข้อ ข้อละ 1 คะแนน = 4 คะแนน
- List 5 ข้อ ข้อละ 0.80 คะแนน = 4 คะแนน
- Tuple 6 ข้อ ข้อละ 1 คะแนน = 6 คะแนน
- รวม 20 คะแนน

## วิธีอัปโหลดขึ้น GitHub Pages

1. เข้าสู่ระบบ GitHub และกด **New repository**
2. ตั้งชื่อ repository เช่น `python-print-data-structure`
3. เลือก **Public** แล้วกด **Create repository**
4. กด **Add file > Upload files**
5. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้:
   - `index.html`
   - `grader-worker.js`
   - `.nojekyll`
6. กด **Commit changes**
7. เข้า **Settings > Pages**
8. ในหัวข้อ **Build and deployment**
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/(root)`
9. กด **Save**
10. เมื่อเผยแพร่แล้ว URL จะมีรูปแบบ:
   `https://USERNAME.github.io/python-print-data-structure/`

## การใช้งานในชั้นเรียน

1. ส่งลิงก์ GitHub Pages ให้นักศึกษา
2. นักศึกษากรอกรหัสและชื่อ
3. เลือกข้อ เขียนโค้ด และกด “ส่งตรวจข้อนี้”
4. ระบบบันทึกงานใน localStorage ของเบราว์เซอร์
5. เมื่อทำเสร็จ ให้นักศึกษากด “ส่งออกผลทั้งหมด”
6. นักศึกษาส่งไฟล์ JSON ที่ดาวน์โหลดได้ผ่าน LMS, Google Classroom หรืออีเมล

## ข้อกำหนดที่ตีความให้ชัดเจน

- ข้อ complex ที่ให้พิมพ์ “ค่าความจริง” ตรวจเป็น `bool(complex1)`
- การตรวจอักขระ `ก` หมายถึงตรวจว่า `ก` อยู่ในข้อความด้วยตัวดำเนินการ `in`
- ช่วง index ระบุแบบรวมตำแหน่งปลาย:
  - String -6 ถึง 4 ได้ `Pytho`
  - String -4 ถึง -2 ได้ `tho`
  - Tuple index 1 ถึง 3 ได้ `('two', 'three', 1)`
- ข้อรับ input ไม่ควรใส่ข้อความ prompt ใน `input()` เพื่อให้ผลลัพธ์ตรงกับ grader
